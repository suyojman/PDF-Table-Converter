# First install the dependencies of the program:
 1. pip install camelot-py[cv]
 2. pip install PyPDF2

# Save the pdf in the working directory where you have kept the converter.py

# Then in the terminal from the working directory type

	python converter.py <nameofthepdf>

	Eg: python converter.py abc.pdf

# Now the program will automatically convert all the table into csv files.
